# Curso Maquetación y Desarrollo Frontend

En este repositorio estaré compartiendo el código que vayamos desarrollando en las clases del curso. Igualmente, ustedes podrán subir su código a carpetas asignadas.

Para subir su código:

1. Hacer un fork del repositorio
2. Clonar el fork a su equipo
3. Crear una nueva rama cuyo nombre sea su nombre con un apellido
4. Hacer commit de sus cambios en esa rama
5. Subir los cambios a GitHub con push
6. En su navegador, crear un pull request AL REPO ORIGINAL
